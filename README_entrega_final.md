# Entregable Final — Gestión de Residuos

Este paquete permite visualizar y demostrar el proyecto sin instalar nada.

## Archivos
- `index_final_offline.html` — Vista de entrega final (sin conexión).
- `index_final_api.html` — Prueba de API (conexión a tu backend).
- `data/seed_final.json` — Datos semilla.
- `assets/logo.svg` — Logo base.

## Uso
1. Abra `index_final_offline.html` para evaluar la demo.
2. Abra `index_final_api.html`, configure la URL base del backend y pruebe GET/POST/PUT/DELETE.
3. El botón **Imprimir/Guardar PDF** en la vista offline permite generar un archivo evaluable.
